Save Your Wallpaper Settings
Your current settings are encoded in the URL below. iOS Shortcuts fetches this daily to generate a fresh wallpaper every morning.

2
Create a Daily Automation
Open the Shortcuts app on your iPhone:
→ Tap Automation tab → + → New Automation
→ Choose Time of Day → 6:00 AM · Repeat: Daily
→ Select "Run Immediately" (no confirmation)
3
Add Two Actions
3.1 Search "Get Contents of URL" → paste URL from step 1
3.2 Search "Set Wallpaper Photo" → choose Lock Screen
⚠ Important: Expand options → disable "Crop to Subject" and "Show Preview".
✓
Done!
Test by tapping ▶ Run in the Shortcuts app.

Daily Auto-Update via cron
—
Paste into Terminal (replace YOUR_URL):
(crontab -l 2>/dev/null; echo "0 6 * * * curl -o ~/Pictures/mindful.png 'YOUR_URL' && osascript -e 'tell app \"Finder\" to set desktop picture to POSIX file \"'$HOME'/Pictures/mindful.png\"'") | crontab -
✓
All Set!
macOS downloads and sets the wallpaper every morning at 6 AM.

PowerShell Auto-Update Script
—
Save as mindful.ps1 (replace YOUR_URL):
$url="YOUR_URL"; $out="$env:USERPROFILE\Pictures\mindful.png"; Invoke-WebRequest $url -OutFile $out; Set-ItemProperty -path 'HKCU:\Control Panel\Desktop' -name Wallpaper -value $out; RUNDLL32.EXE user32.dll,UpdatePerUserSystemParameters
3
Schedule with Task Scheduler
Task Scheduler → Create Basic Task → Daily 6 AM
Action: powershell.exe -ExecutionPolicy Bypass -File "C:\path\mindful.ps1"
✓
All Done!
Right-click the task → Run to test immediately.
