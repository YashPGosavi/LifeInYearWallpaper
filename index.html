Auto-Update Setup
Automatically refresh your lock screen wallpaper every morning at 6 AM.
1
Save Your Wallpaper Settings
Your current settings are encoded in the URL below. This is what iOS Shortcuts will fetch daily — it auto-generates your wallpaper fresh every morning.
—
💡 To make this URL work, open the HTML file on a local server or host it on GitHub Pages / Vercel. The URL above will work when hosted.
2
Create a Daily Automation
Open the Shortcuts app on your iPhone:
→ Tap Automation tab at the bottom
→ Tap + → New Automation
→ Choose Time of Day → set to 6:00 AM
→ Set Repeat to Daily
→ Select "Run Immediately" (no confirmation)
→ Tap "Create New Shortcut"
3
Add Shortcut Actions
Inside the new shortcut editor, add these two actions in order:
3.1 Search for "Get Contents of URL"
→ Paste the URL you copied in step 1
3.2 Search for "Set Wallpaper Photo"
→ Choose Lock Screen
⚠ Important: In "Set Wallpaper Photo", tap the arrow → to expand options → disable both "Crop to Subject" and "Show Preview". This prevents iOS from cropping or asking for confirmation every day.
✓
Done!
Your wallpaper will now auto-update every morning at 6 AM with fresh progress data. Test it by tapping ▶ Run in the Shortcuts app.
1
Save Your Settings URL
MacroDroid will fetch this URL daily to generate your wallpaper image.
—
2
Install MacroDroid
Download MacroDroid from the Google Play Store. It's free and lets you automate wallpaper changes on a schedule.
3
Create a Macro
Open MacroDroid → tap Add Macro
Trigger: Date/Time → Day/Time → 00:01:00
→ Enable all weekdays
4
Configure Two Actions
4.1 Download Image
Web Interactions → HTTP Request
Method: GET · Paste the copied URL
✓ Enable "Block next actions until complete"
✓ Tick "Save HTTP response to file"
Filename: /Download/mindful.png
4.2 Set Wallpaper
Device Settings → Set Wallpaper
Enter same filename: /Download/mindful.png
Choose Lock Screen
⚠ Important: Use the exact same folder and filename in both actions, otherwise the wallpaper won't update.
5
Finalize & Test
Give your macro a name → tap Create Macro.

To test: MacroDroid → Macros → select your macro → More options → Test macro
Updates daily at 6:00 AM
